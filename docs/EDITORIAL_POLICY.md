# 編集方針・中立性ポリシー

このダッシュボードとオープンデータは、議会広報の延長として「誰が・いつ・何を質問し、町がどう答えたか」を**事実として**伝えることを目的とする。評価・採点・順位付けは目的としない。

## 1. 質問数を序列化しない
一般質問の「数」は努力指標であって成果指標ではなく、議員間の比較に用いることはできない。理由は次のとおり。

- **在任期間が違う**（補欠当選等で登壇できた定例会の回数が異なる）。
- **登壇機会が一様でない**（体調等で登壇しない期間がある）。
- **数え方が標準化されていない**（1論点を①②③と分けて問う議員と、1問にまとめる議員では件数が大きく変わる）。

したがって、既定表示は**議席番号順**とし、件数による棒グラフのランキングや「多い順」「割合（％）」の強調は行わない。件数は各議員カードに**事実として小さく**示すにとどめる。

## 2. 並べ替えは「ユーザーの操作」として開く
利用者が自分の関心で並べ替え・絞り込みをするのは自然なので、機能としては提供する。ただし
- 初期表示は中立な並び（議席順）とする。
- 件数等で並べ替えた場合は、上記1の注意書きを**自動的に併記**する。

序列を見せるのではなく、利用者が探索する道具を提供する、という立場を保つ。

## 3. テーマ分類は「広がり」を見るためのもの
各質問は主たる政策領域で1テーマに分類した**実務的な区分**であり、境界事例は判断による（恣意性がある）。テーマは件数比較ではなく、扱われた論点の広がりを示すために用いる。分類基準は広報委員会・議員に確認のうえ改訂しうる。

## 4. 一次情報への動線を常に確保する
議会だより掲載文は**編集・要約された版**である。全文は会議録・録画にあたれるよう、各質問から会議録／録画／議会だより（PDF）へのリンクを必ず提供する。未掲載の質問（3問以上で〇の付かなかったもの）は、題名のみ一覧表（目次）に載るため、本文ページではなく一覧表を参照先とし、本文は会議録で確認できる旨を明示する。

## 5. 出典・著作権・個人情報
議会だよりは町（議会）の著作物である。転載・再配布時は出典を明記し、利用規約に従う。データには出典とライセンス（コード=MIT／データ=CC BY 4.0）を併記する。個人情報は議会だより・会議録に既に公開された範囲を超えて収集しない。

## 6. 選挙期間での利用
現職の活動実績を扱う性質上、党派的な表現・優劣の含意を避け、事実提示と一次資料への動線に徹する。選挙利用・転載の可否は、最終的に議会事務局・選挙管理委員会に確認する。
