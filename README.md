# council-dashboard 北海道森町版データ

[kken78/council-dashboard](https://github.com/kken78/council-dashboard) を北海道茅部郡森町（もりまち）にローカライズするための差し替えデータです。元リポジトリの `config.json` と `data/` をこのフォルダの内容に置き換えて `python build.py` を実行すると、森町版のダッシュボードが生成されます。

## 使い方

1. 本フォルダの `config.json` を、council-dashboard リポジトリの `config.json` に上書き。
2. 本フォルダの `data/issue-115.json` 〜 `data/issue-125.json` を、リポジトリの `data/` に配置（元の日野町の `issue-24〜35.json` は削除してよい。`_template.json` は残す）。
3. `python scripts/validate.py` で検証 → `python build.py` で `docs/` を生成。

## 作成上の判断・注意点（要確認）

森町の議会だよりは画像（アウトライン化）PDF でテキスト抽出ができないため、PDF を目視で読み取り転記しています。重要箇所は原文 PDF と照合してください。

- **第11議席**：令和6年9月（第120号）までは **山本 裕子** 議員が在職。その後辞職し、令和7年6月現在は空席です。`config.json` では seat 11 を山本議員＋辞職注記としています。ふりがな（やまもと ゆうこ）は推定です。
- **緊急質問は未収録**：第116号には河野文彦議員の緊急質問「酷暑から命を守る対策について」が掲載されていますが、本データは「一般質問」に限定したため収録していません。
- **題名のみの質問（published:false）**：第118号（松田議員の2件）はスペースの都合で題名のみ掲載のため、`published:false`・題名のみで収録しています。
- **テーマ分類**：元の日野町版のテーマ区分を踏襲し（「農林業」→森町に合わせ「農林水産業」に変更）、各質問を主たる政策領域で1テーマに分類しています。境界事例は判断によります。
- **答弁者ラベル**：森町は二元代表制で、答弁は主に「岡嶋 町長」「毛利 教育長」。1問に複数答弁者がいる場合は `a[]` に複数ブロックで収録しています。
- **発行日**：`published_date` は各号の発行月の1日を便宜的に設定しています（PDF に発行日（日）の明記がないため）。
- **議席番号**：議会だより索引上の議席番号に一部誤植が見られたため（例：山田議員を「6番」表記の号あり）、`config.json` の seat は森町公式「議員名簿」に従っています。
